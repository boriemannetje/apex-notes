Graph interactions

  • Click empty graph space in an empty folder to create the first note.
  • Double-click empty graph space to create a child note, then drag the purple rope onto a parent node.
  • Drag a dashed loose node onto another node to connect it as a child.
  • Click a node to open it. Shift-drag, Command-drag, or Control-drag empty space to select several nodes.
  • Drag selected nodes to move them. Press Backspace or Delete to move selected notes to Trash.
  • Copy, cut, and paste selected nodes. Paste text into the graph to create a note, or drop Markdown files to import them.
  • Scroll to zoom. Alt-drag or middle-drag to pan. Use Reset to fit the graph.
